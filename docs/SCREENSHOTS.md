# Screenshots

This page showcases the GiwiCD user interface.

## Landing Page

![Landing Page](images/landing.png)

The landing page features a modern hero section with quick access to sign in, a visual pipeline mockup, and an overview of key features.

## Login

![Login](images/login.png)

The login page provides a clean authentication form with email and password fields. New users can register if enabled by an admin.

## Dashboard

![Dashboard](images/dashboard.png)

The dashboard displays real-time statistics including total builds, success rate, and recent build history. All data updates automatically.

## Pipelines

![Pipelines](images/pipelines.png)

The pipeline management page shows all configured pipelines with their status, last build information, and quick actions to run, edit, or delete.

## Builds

![Builds](images/builds.png)

The builds page lists all build history with status indicators, duration, and trigger information. Click any build to view detailed logs and stage progress.

## Credentials

![Credentials](images/credentials.png)

The credentials manager securely stores SSH keys, tokens, and notification credentials. Credentials can be used for private repository access or build notifications.

## Settings

![Settings](images/settings.png)

The admin settings page allows configuration of system-wide options including max concurrent builds, build retention policies, and registration controls.
